# Git Class Task 1
setup git on your system with all the required tools in mind

### Tasks
- verify that you have git install
    - if not, install it with package manager of your system
    - check the installed version
    - compare installed version to the latest version on the git project website
    - if too many version apart:
        - remove installed version
        - download the latest version source code
        - install all dependencies for building the binary locally
        - compile git for local use
        - verify that latest version is indeed installed
        - check the location
- configure you global git configuration 
- create new project called ~/Projects/git-shallow-dive
    - create local git configuration with different username
